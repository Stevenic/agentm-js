## Here's the contents of the styles.css file

### Classes:

- body: Applies general styles for the entire body, including background color (#1e1e1e), font settings, and setting up a flexbox layout for the content.
- chat-panel: Styles the chat section on the left, setting its width, background color, padding, and display as a flex container to allow vertical stacking of elements.
- chat-header: Styles the chat header area with a font size, padding, background color, and centered text.
- chat-messages: Defines the message display area with scrollable content (overflow-y: auto), padding, and background color. This section grows to fill available space.
- chat-message: Styles individual chat messages with padding, margin, background color, and rounded corners.
- form: Ensures that form elements (input and submit button) are stacked vertically and fill the full width of the panel.
- chat-input: Styles the message input field with padding, background color, and full-width layout. Includes margin for spacing between the input and the button.
- chat-submit: Styles the submit button with padding, background color, cursor pointer, and full-width layout. It also includes a hover effect to change the background color when the button is hovered.
- viewer-panel: Styles the right-hand viewer panel, setting its width, background color, padding, and center-aligning its contents both horizontally and vertically using flexbox.
- loading-overlay: Styles the loading page shown when a message is submitted.
- spinner: Styles the loading spinner.
